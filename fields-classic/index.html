수업 진행 방식

1단계

온라인 진단 모의고사

  • 결제 후 제공된 승인번호 입력 후 응시 가능
  • 실제 시험 유사 레벨 테스트를 통해 현재 아이의 정확한 수준 파악
  • 오프라인 응시도 가능
2단계

온라인 진단 분석지

  • 영역별 강점과 약점을 세부적으로 분석하여 개인별 진단표 작성
  • 목표 레벨과 현재 실력 간의 격차를 수치화하여 명확한 목표 설정

3단계

풀이 영상

  • 응시한 테스트의 풀이 영상으로 부족 유형 완전 학습
  • 부족한 유형에 대한 이해도 체크

맞춤형 학습 시스템

개인별 학습 계획

진단 결과를 바탕으로 아이의 현재 수준과 목표에 맞는 맞춤형 커리큘럼을 설계합니다. 약점은 보완하고 강점은 더욱 발전시키는 전략적 접근법을 적용합니다.

약점 집중 보완

개인별로 드러난 취약 영역에 대해 집중적인 보완 학습을 진행합니다. 단순 문제 풀이가 아닌 개념부터 탄탄하게 이해할 수 있도록 지도합니다.

실전 문제 훈련

실제 시험과 유사한 환경에서의 문제 풀이 훈련을 통해 시간 관리 능력과 문제 해결 전략을 체득하도록 합니다.
